Datos extraidos del Banco Mundial para el desarrollo del examen.
El obejtivo del trabajo es usar indicadores del Banco Mundial para prepara un dataset y entrenar modelos que permiten predecir o clasificar el nivel del PIB de los paises. Para lograrlo, se realiza limpieza e imputacion de datos, se aplica PCA para reducir variables y finalmente se entrenan y evaluan modelos de clasificacion.
