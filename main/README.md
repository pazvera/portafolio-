Datos extraidos del Banco Mundial para el desarrollo del examen.
El obejtivo del trabajo es usar indicadores del Bnaco Munidla para prepara un dataset y entrenar modelos que permten predecir o clasificar el nivel del PIB de los paises. Para lograrlo, se realiza lipieza e imputacion de datos, se aplica PCA para reducir variables y finalmente se entrenan y evaluan modelos de clasificacion.
